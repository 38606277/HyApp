# vant-store
一个商城Demo, 前端使用vant有赞UI,vue实现, 后端使用mysql,express实现。

# 实现的功能

1. 登录/注册
2. 商品列表
3. 商品详情
4. 购物车
5. 收货地址
6. 订单
